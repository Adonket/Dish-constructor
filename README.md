Название проекта: Dish Designer  
Цель: создать приложение для планирования питания, которое минимизирует время на принятие решений, сокращает пищевые отходы и помогает 
пользователям питаться разнообразно и в соответствии с их предпочтениями.\n
Задачи:  
1. Внедрить добавление PDF-чеков, либо фотографий чеков покупок, для 
пополнения базы данных имеющихся продуктов. 
2. Реализовать планирование своего рациона на несколько дней 
3. Внедрить формирование списка покупок продуктов для приготовления 
блюд 
4. Обеспечить детальную персонализацию: диетические ограничения, 
предпочитаемые кухни, аллергии, бюджет, цели (похудение, поддержание веса, 
набор массы)

Код соответствует основному стилевому руководству для Python - PEP 8
